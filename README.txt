TERMINAL UI SITE (original)

1) 使い方
- このフォルダをそのままGitHubリポジトリのルートに置く
- GitHub Pages を有効化

2) ファイル
- index.html   : 起動ログ→メニュー
- files.html   : アーカイブ
- terminal.html: コマンド入力
- about.html   : 仕様
- style.css    : デザイン
- app.js       : 共通JS

3) 編集ポイント
- files.html のカード一覧は app.js の FILES_DB を編集
- 色は style.css の :root 変数

4) 公開時の注意
- 公式ロゴ/公式画像/固有名詞を避け、抽象要素でモチーフ化すると安全
